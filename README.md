## 🤝 Collaborative Git Workflow

To ensure smooth collaboration and avoid file conflicts, please follow this workflow when updating your page:

1. Always pull the latest changes before starting: `git pull origin main`
2. Create your own feature branch: `git checkout -b feature/your-name`
3. Stage and commit your updates: `git commit -am "feat: update page layout"`
4. Push your branch online: `git push origin feature/your-name`
5. Open a Pull Request on GitHub for review.
